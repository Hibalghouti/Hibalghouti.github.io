Usage Guide

This calculator supports evaluating prefix and postfix expressions. Use the respective input fields to enter the expressions and click the relevant button to compute the result.

Addition
-
Subtraction
*
Multiplication
/

Examples

Prefix

  • + 3 5 (Result: 8)
  • * + 2 3 4 (Result: 20)
  • + 1 * 2 3 (Result: 7)

Postfix

  • 3 5 + (Result: 8)
  • 2 3 + 4 * (Result: 20)
  • 1 2 3 * + (Result: 7)
